Space Odyssey Website
A simple, responsive website about space exploration milestones.

Description
This project is a single-page website showcasing key events in space history. It features a clean design with a dark space-themed background and responsive layout.

Features
Responsive design for all devices

Space-themed color scheme

Six milestone cards with dates

Simple HTML/CSS structure

Usage
Open index.html in any web browser

No installation or dependencies required

Files
index.html - Main website file

No additional files needed

Customization
Edit the HTML file directly to:

Change text content

Update image URLs

Modify colors in the style section
